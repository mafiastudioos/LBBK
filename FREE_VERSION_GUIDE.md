## 🛠️ **Setup Instructions**

### **Step 1: Use Free Chatbot**
Replace the AI chatbot with the rule-based version:

```bash
# In src/App.js, replace:
import ChatBot from './components/ChatBot';

# With:
import FreeChatBot from './components/FreeChatBot';

# And replace:
<ChatBot />

# With:
<FreeChatBot />
```

### **Step 2: Set up Free Email (Optional)**
1. Create a Gmail account for your business
2. Enable "App Passwords" in Gmail settings
3. Add to your .env file:
```
BUSINESS_EMAIL=your-business@gmail.com
EMAIL_APP_PASSWORD=your-app-password
```

### **Step 3: Deploy for FREE**
```bash
# Deploy to Vercel (free)
npm install -g vercel
vercel

# Or deploy to Netlify (free)
npm install -g netlify-cli
npm run build
netlify deploy --prod --dir=build
```

### **Step 4: Get Free Domain**
1. Go to [Freenom.com](https://freenom.com)
2. Search for: `lashbeautybykim.tk`
3. Register for free (12 months)
4. Point DNS to your Vercel/Netlify URL
